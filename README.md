## Execução (ambiente Windows)

1.  **Compile o projeto:**

    ```bash
    gcc main.c arquivo.c arvoreb.c -o main
    ```

2.  **Execute o projeto:**

    ```bash
    ./main.exe
    ```

## Extra (opcional)

**Rode o seguinte comando para preencher os arquivos com dados de teste:**

    gcc seeder.c arquivo.c arvoreb.c -o seeder.exe; ./seeder.exe
